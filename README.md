## Sway config

SwayWM `stow` package.

```bash
git clone https://github.com/tehlastjuan/swaywm && stow --stow swaywm/
```

#### main config

```bash
./swaywm/bash
./swaywm/bluetuith
./swaywm/environment.d
./swaywm/fontconfig
./swaywm/foot
./swaywm/gtk-3.0
./swaywm/kanshi
./swaywm/kvantum_themes
./swaywm/mako
./swaywm/qt5ct
./swaywm/qt6ct
./swaywm/sway
./swaywm/systemd # check matching files in /etc/systemd/user
./swaywm/terminfo
./swaywm/waybar
./swaywm/wob
./swaywm/xdg-desktop-portal
```

#### scripts

```bash
./swaywm/bin
./swaywm/share/bash_lib/sway
./swaywm/share/bash_lib/utils
```

I hope you find something useful for your config.
